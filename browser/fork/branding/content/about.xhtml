AnonNet Browser

Version 1.0.0
A privacy-focused browser built on Firefox ESR with Tor Browser hardening, designed exclusively for anonymous browsing on the AnonNet network.
Base
Firefox ESR 128
Network
AnonNet
Proxy
SOCKS5 (Locked)
Domain
.anon only

Features

  • Tor Browser privacy hardening
  • Fingerprinting resistance (RFP)
  • Built-in credit monitoring
  • Clearnet access blocked
  • DNS leak protection
  • WebRTC disabled
  • All telemetry disabled
  • Automatic proxy configuration
⚠️ Security Notice: This is experimental software. Do not use for activities requiring strong anonymity guarantees. Your anonymity depends on the AnonNet network security.
